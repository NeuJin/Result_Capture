# Conrod Analysis Post-Processing — Setup Guide

## Quick Start

### Option A: Auto-build (recommended)

**Requires:** Python 3.8+ (standard library only — no pip install needed)

```
python build_xlsm.py
```

Before running, enable VBA project access in Excel:
> File → Options → Trust Center → Trust Center Settings
> → Macro Settings → tick **"Trust access to the VBA project object model"**

### Option B: Manual import

1. Open Excel, create a new workbook, save as `.xlsm`
2. Press `Alt+F11` to open VBA Editor
3. `File → Import File`, import each file from the `vba\` folder in order:
   - `ConfigManager.bas`
   - `InpParser.bas`
   - `TCLTemplates.bas`
   - `HVRunner.bas`
   - `ReportStager.bas`
   - `PPTXBuilder.bas`
   - `SetupWorkbook.bas`
4. In the Immediate Window (`Ctrl+G`), type:
   ```
   SetupWorkbook.InitialiseWorkbook
   ```
   and press Enter

## Usage

1. **Config sheet** — fill in:
   - Result file path (`.odb`, `.res`, `.h3d`, `.op2`)
   - Model `.inp` file path (Abaqus input deck with `*NSET`)
   - Analysis Type: `Stress` / `SafetyFactor` / `Both`
   - HyperView exe path (e.g. `C:\Altair\2022\hw\bin\win64\hw.exe`)
   - Output folder

2. **Click [Parse .inp]** — reads `*NSET` sections from the model file, fills NodeSets sheet

3. **NodeSets sheet** — review parsed sets, set `Include = Y/N`, edit display labels

4. **Legend sheet** — set Min/Max/Levels etc. (never auto-adjusted by the tool)

5. **Views sheet** — choose which of the 6 views to capture, auto-fit ON/OFF

6. **Click [Run Analysis]** — generates TCL, launches HyperView, fills Report sheet

7. **Report sheet** — review all screenshots, edit captions if needed

8. **Click [Generate PPTX]** — exports PowerPoint report to `output\reports\ConrodReport.pptx`

## HyperView API Verification (Phase 4 prerequisite)

Before running the full analysis, verify these TCL commands in HyperView's TCL console:

```tcl
hwi GetSessionHandle s
s GetProjectHandle p
p GetPageHandle pg 0
pg GetWindowHandle w 0
w GetClientHandle c
w GetModelHandle m 0

# Test mask/unmask
m SetMaskState 0   ; # hide all
m SetMaskState 1   ; # show all

# Test view + fit + capture (replace numbers with actual screen size)
c SetViewPoint 1 1 1 0 0 1
c Fit
hwi CaptureScreen "C:/temp/test.png" 1920 1080
```

If any command errors, check the HyperView 2022 TCL Reference Guide and update
the corresponding token in `TCLTemplates.bas > GetScreenshotBlock()`.
